# CodeCoach AI

CodeCoach AI 是一个面向 Java 后端实习 / 校招求职者的 AI 面试教练平台。

项目第一版目标是完成“项目拷打训练闭环”：

用户录入项目描述 -> AI 生成面试追问 -> 用户回答 -> AI 继续追问 -> 生成训练反馈。

## 项目阶段

当前阶段：项目立项与需求分析。

## 技术栈，暂定

- 后端：Java 21、Spring Boot 3、MyBatis-Plus、MySQL、Redis
- 前端：React、TypeScript、Vite、Ant Design
- AI：外部大模型 API
- 部署：Docker Compose、Nginx
